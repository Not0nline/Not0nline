## Hi there, I'm Adrial Natanael Liong! 👋

### Systems & Security Engineer — Windows Kernel · Rust · C · EDR

I build low-level security software. Right now I develop the **kernel-mode core of a Windows EDR** (Endpoint Detection & Response) — a WFP network filter driver, a kernel↔user-mode telemetry pipeline, and a host-isolation engine — paired with a **Rust** user-mode agent. I'm most at home close to the metal, but I work comfortably up the stack through backend microservices and QA.
